# FlowTools

A DaVinci Resolve Studio Workflow Integration with focus on automation and tools for video creation.

FlowTools is developed by Zlice. 
AI-assisted development tools are used for parts of coding, brainstorming and documentation. All architecture, testing and product decisions are reviewed by the developer.

## Features
- Automation system
- Custom automation editor
- Keybind support
- Overview dashboard panels
- Timeline/video automation tools
- Import/export automations
- Save system with persistent settings

<br>

## Installation
**Note** This integration only works with the studio version of DaVinci Resolve.

1. Download [FlowTools-latest](https://github.com/Zlicen/FlowTools/releases/)
2. Run installer as Administrator
3. Restart DaVinci Resolve
4. Open Workspace → Workflow Integrations → FlowTools
